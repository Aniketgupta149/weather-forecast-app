About

Weather App is a modern, responsive web application that provides real-time weather information for any city worldwide. It uses the OpenWeatherMap API to fetch current weather data and displays it in a clean, user-friendly interface.

Key Features

  • Live weather: temperature, humidity, wind, and sky condition
  • Automatic local time for the selected city
  • 5-day detailed forecast with icons
  • Interactive map: click to view weather anywhere
  • Fully responsive, modern UI for all devices

Tech Stack

  • HTML5, CSS3, JavaScript (ES6+)
  • Bootstrap 5
  • OpenWeatherMap API
  • Leaflet & OpenStreetMap
